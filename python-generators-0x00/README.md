# Python Generators for Database Streaming

## Project Overview
This project demonstrates how to use Python generators to efficiently stream data from a MySQL database. It includes:
1. Database setup and seeding
2. A generator function to stream rows one-by-one

## Setup Instructions

1. **Prerequisites:**
   - MySQL Server installed
   - Python 3.x
   - `mysql-connector-python` package (`pip install mysql-connector-python`)

2. **Database Setup:**
   ```bash
   python3 seed.py
